# Chip8 Emulator and Debugger
Emulator and debugger for Chip8. This project was developed using Python and Pygame.
To use this emulator, download Chip8 ROM from somewhere such as here:
https://github.com/kripod/chip8-roms

Download the Python files and run this command on cmd:
> python main.py [filepath-to-ROM]

To run in Debug mode, run like this:
> python main.py [filepath-to-ROM] DEBUG
